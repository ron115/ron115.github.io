Serialization

  • Although because of a hidden proxy object added by the entity framework, the serialization of DTOs can not be done in asp.net mvc, it can, however, be performed by a customized function using XmlSerializer and MemoryStream.
  • To prevent circular reference, some DTO propoerties a marked as "[XmlIgnoreAttribute]"
